# 电影院购票系统

## 设计模式

1. 单例模式
2. MVC（Model-View-Controller）模式
3. 表现层（Controller），业务逻辑层（Service），和数据访问层（DAO）

## 设计优化

1. 类图简化
2. 数据库
3. 可视化界面

## 数据库连接

在项目文件中的config.properties文件中修改数据库参数，并将数据库文件cinemadb.sql导入本地，默认端口3306

**数据库文件与类图图片放在项目文件中**  
与config.properties文件同级
